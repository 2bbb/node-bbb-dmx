# bit by bit (for node)

## At first

```
npm install bit-by-bit
```

```
const bbb = require('bit-by-bit');
```

## API

### functional

#### range

```
const range = bbb.functional.range;
const arr1 = range(8);    // [0, 1, 2, 3, 4, 5, 6, 7]
cosnt arr2 = range(2, 8); // [2, 3, 4, 5, 6, 7]
const arr3 = range(2, 8, 3); // [2, 5]
```
